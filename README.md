# Pembelajaran-C-fundamental
Fundamental Repositori ini berisi kumpulan materi dan contoh kode dasar bahasa C. Fokus utama meliputi struktur program C, tipe data, variabel, operator, kontrol alur (if, loop), fungsi, serta manipulasi array dan pointer. Cocok sebagai referensi awal bagi pemula yang ingin memahami konsep fundamental pemrograman C secara praktis dan terstruktur.
